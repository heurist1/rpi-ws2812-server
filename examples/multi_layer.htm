This example first draws a red-green picture on the main background, then initializes a new layer with paint operator CAIRO_OPERATOR_DEST_IN and paints text on it.
The CAIRO_OPERATOR_DEST_IN will display the main background image where the text is.
Requires a 32x8 WS2812 LED panel.

setup 1,256,3 init brightness 1, 32 config_2D 1,32,8,3,32,8 draw_image 1,examples/red-green.png init_layer 1,1,8,0,1,0 change_layer 1,1 print_text 1,0,8,"HELLO WORLD", FFFFFF,8,1,0,"enhanced_led_board-7.ttf" render
