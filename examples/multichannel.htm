Add a LED string of 300 LEDs WS2812 chip on PWM0 (GPIO 12) output of the Raspberry Pi.
Add a LED string of 50 LEDs WS2812 on PWM1 (GPIO 13) output.
Add a LED string of 30 SK9822 to SPI 1 interface MOSI and CLK.
Execute these commads to fill first string with RED, second with GREEN and third blue.

setup 1,300,3 setup 2,50,3,0,255,19 setup 3,30,12 init fill 1,FF0000 fill 2,00FF00 fill 3,0000FF render 1 render 2 render 3

Use CTRL+C key to stop the rotating and return to command line.
